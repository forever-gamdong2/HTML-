form 태그의 속성
  • action 속성 : 서버 URL 경로 표시
  • method 속성 : GET-공개용이나, POST-비공개용 이냐로 나누어 전달
  • cf) GET 방식으로 서버에 전송된 것은 주소표시줄에 경로가 전부 노출됨
  • cf) POST 방식으로 서버에 전송된 것은 주소표시줄에 경로가 끊겨보임 or 노출안됨
input 태그의 속성
  • name 속성(필수는 x) : 해당 타입의 목적을 띈 이름을 작성 ex) radio 버튼이 'menu'의 목적이 있음을 밝힘
  • value 속성(필수는 x) : 초기값
  • id 속성 : label 태그와 같이 사용됨. JS의 연결점이 되기도 함
