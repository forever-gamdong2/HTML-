01 다음 괄호 안을 채우시오.

  • ( 입력양식; form )은 사용자에게 정보를 입력받는 요소입니다.
  • form 태그는 ( method )속성의 방식으로 ( action )속성 장소에 데이터를 전달합니다.
  • ( get )방식은 주소에 데이터를 입력해서 전달합니다.
  • ( post )방식은 GET 방식과 달리 비밀스럽게 데이터를 전달합니다.
  • input 태그에 ( type )속성을 지정해서 다양한 종류의 기본 입력 양식을 생성합니다.
  • 라디오 버튼의 ( name, checked )속성을 사용하면 여러 대상 중 하나만 선택하는 형태를 구현할 수 있습니다.
  • ( select )태그는 목록으로 보여 주는 항목 중 하나 또는 여러 개를 선택할 때 사용하는 입력 양식 요소입니다.
  • div 태그는 공간을 ( block )형식으로 분할합니다.
  • span 태그는 공간을 ( inline )형식으로 분할합니다.
  • ( 시멘틱, 시멘틱 웹, 시멘틱 태그 )은 컴퓨터 프로그램이 코드를 읽고 의미를 인식할 수 있는 지능형 웹을 의미합니다.


02 다음 질문에 OX를 표시하시오.

  1. “https://www.google.com/search?q=초콜릿”처럼 주소에 “q=초콜릿” 데이터를 넣은것은 POST 방식의 요청입니다. ( X; get )
  2. label 태그의 for 속성에 “입력 양식 태그의 id 속성”을 넣으면, label 태그를 선택했을 때입력 양식에 포커스가 갑니다. ( O )
  3. select 태그 내부에 opt 태그를 넣어서 선택할 수 있는 요소를 만듭니다. ( X; option)
  4. 현대 웹 페이지의 레이아웃을 구성할 때는 테이블을 사용합니다. ( X; 시멘틱박스 )
  5. a 태그와 i 태그 등 텍스트는 블록 형식의 태그입니다. ( X; a-inline / i-꾸밈요소 )
  6. 모든 검색 엔진은 웹문서에 적용된 스타일을 시각적으로 분석하고 의미를 파악합니다. ( X; "시각적으로"는 아님 )
  7. 시맨틱 태그는 시맨틱 웹을 구현할 때 사용하는 방법입니다. ( O )
